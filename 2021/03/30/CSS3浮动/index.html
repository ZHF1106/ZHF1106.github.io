display属性

块元素（header / section / article / footer / p / table / h1-h6 / div / nav）

1、独占一行
2、默认宽度是100%
3、可以设置宽高

内联元素（span / strong / em / label / a / input / select / img）

1、不独占一行
2、满一行自动换行
3、不能设置宽高

混合模式

1、对象还是按内联元素排布
2、对象的内容按块元素设置
3、既不会独占一行，又可以设置宽高

隐藏元素：display属性设置为none

float浮动

float与块元素一样都能显示一行占用

默认值，不浮动

元素向左浮动

元素向右浮动

浮动的本质

1、脱离原有的标准文档流
2、根据方向位于父元素的左上角或右上角
3、多个浮动的元素会按照流式布局依次排列下去

案例

清除浮动

clear属性清除浮动

默认值，不清楚浮动

清楚左浮动

清楚右浮动

清楚左右浮动

overflow属性清楚浮动

默认值，显示超出部分

隐藏超出部分

显示滚动条，可利用滚动条查看超出部分

自动，超出的情况才显示滚动条

after伪类元素清除浮动

1、在该元素的所有子元素的末尾添加一个节点
2、通过content来设置节点的文本内容
3、content属性不能省略，没有内容，也要设置为空字符串
4、清除浮动时设置clear，display设置为为块元素
5、可以将after改为befor，用法一样，只是在子元素的开头添加一个节点

inline-block和float的区别

inline-block

1、元素一行显示，可以设置宽高
2、不会脱离标准文档流，不用清除浮动，代码实现方便
3、位置方向不可控制，只能从左到右
4、元素之间会有空格，需要把字体大小设置为0
5、老版本浏览器（IE6/7）不支持此属性

float

1、让元素一行显示，可以设置宽高
2、可以决定排序方向，左浮动或者右浮动
3、脱离标准文档流，对其他元素有影响
4、必须清除浮动，用的比较多
